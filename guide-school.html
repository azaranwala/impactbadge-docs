School Admin & Teacher Guide

Manage student compliance, track graduation hours, and run reports

Two Roles — School Admin vs Teacher

🏫 School Admin (Counselor)

  • Full access to all dashboard features
  • Set graduation hour requirements
  • Approve/reject pre-approval requests
  • Bulk import student rosters
  • Configure school settings
  • Generate district reports
  • Manage Clever Sync connection
  • Register school domains
  • Generate Partner API keys

👩‍🏫 Teacher (Advisor)

  • Read-only student list with hours
  • View compliance status
  • Export student data as CSV
  • See school-wide summary stats
  • Cannot change requirements
  • Cannot bulk import
  • Cannot access settings
  • Cannot verify hours

Setting Graduation Requirements

1. Go to Settings

School Dashboard → Settings page.

2. Set Graduation Minimum Hours

e.g., 75 hours for graduation, 10 hours per grade per year.

3. Set Deadlines

Senior deadline (e.g., May 1), quarterly checkpoints.

4. Configure Categories

Which volunteer categories count toward graduation (CSL categories for HCPS).

5. Save

Settings are saved and apply immediately. Students see their progress bar update.

Settings Persist: When you return to the Settings page, your saved values are pre-loaded automatically.

Monthly Graduation Nudges (Automatic)

On the 1st of every month, ImpactBadge automatically:

  1. Checks every student's progress against graduation requirements
  2. Sends push notifications to at-risk and below-minimum students
  3. Notifies linked parents of behind students
  4. Emails the school admin a compliance digest with counts and a CSV attachment

Clever Sync (Roster Management)

If your district uses Clever Complete, student and teacher rosters sync automatically:

  • Daily full sync at 2:00 AM EST — students, teachers, sections, schools
  • Every 15 minutes — incremental changes (new students, class transfers, removals)
  • Removals — students removed from Clever are archived (not deleted). Their hours are preserved.
  • Restorations — if a student reappears in Clever, they're automatically restored with all historical data.

View sync status at /school/clever — see counts, last sync time, and trigger a manual sync.

Bulk Student Import (CSV)

Import your student roster without Clever:

  1. Prepare a CSV with columns: Email, Name, Grade (optional), Student ID (optional), Graduation Year (optional)
  2. Go to Import Students → upload the CSV (max 500 rows)
  3. Existing students are updated; new students are pre-provisioned (applied when they sign up)

Pre-Approval Workflow (HCPS CSL)

1. Student Submits Request

Student fills in: organization name, planned activity, estimated hours, planned dates.

2. Admin Reviews

Go to Approvals → see pending pre-approvals. Review the organization and activity.

3. Approve or Reject

Approve to let the student proceed, or reject with a comment explaining why.

4. Student Notified

Student receives a notification with the decision. Approved students can begin volunteering.

Teacher Advisor Dashboard

Teachers see a simplified read-only dashboard at /advisor:

  • Stats cards: total students, total hours, on-track percentage
  • Searchable student list with hours, grade, and compliance status
  • Export student data as CSV
  • No write access — cannot modify settings, import students, or verify hours

LMS Connect (Canvas / Schoology)

If your district has Clever LMS Connect enabled:

  1. Teachers can create volunteer hour assignments that appear in the LMS gradebook
  2. Students complete hours in ImpactBadge
  3. Grades auto-sync — hours are converted to points: (hours completed / required hours) × points possible
  4. Students can launch ImpactBadge directly from Canvas/Schoology via LTI 1.3
